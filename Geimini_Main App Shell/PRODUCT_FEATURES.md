# Attonomous Product Feature List

This document outlines the core functionality of the platform, organized by screen and prioritized for development teams and stakeholders.

## 1. Home (Command Center)
| Feature | Priority | Description |
|:---|:---|:---|
| **Hub Navigation** | P0 | Unified entry point to all sub-studios. |
| **KPI Cards** | P0 | Glancable metrics (Approvals, Active Use Cases) per functional area. |
| **Platform Status** | P1 | Visibility into engine and API health. |
| **Global Search** | P2 | Search for specific use cases or campaigns across the platform. |

## 2. AI Decisioning (Strategy Studio)
| Feature | Priority | Description |
|:---|:---|:---|
| **Use Case Tracking** | P0 | Monitor incremental lift and budget burn for all RL strategies. |
| **Strategy Setup Wizard** | P1 | Guided flow for audience and guardrail definition. |
| **Lift Comparison** | P1 | Compare RL performance against historical control baselines. |
| **Decision Explainability** | P2 | View logic behind individual delivery vs. holdout decisions. |

## 3. Campaigns (Execution Hub)
| Feature | Priority | Description |
|:---|:---|:---|
| **Budget Controller** | P0 | Global control for daily spend limits and allocations. |
| **Performance Review** | P0 | Analysis of yesterday's execution across all partners. |
| **Automated Bundling** | P1 | One-click selection and scheduling of tomorrow's multi-channel runs. |
| **AI Insights** | P2 | Narrative explanations of performance drivers. |

## 4. Analytics (Insight Engine)
| Feature | Priority | Description |
|:---|:---|:---|
| **Incremental Lift Graph** | P0 | Timeline visualization of value added by the AI engine. |
| **Partner Contribution** | P1 | Attribution of volume and spend to specific partners (e.g. BFL, SBI). |
| **Channel ROI** | P1 | Comparison of cost-per-conversion across communication channels. |

## 5. Content Studio (Creative Lab)
| Feature | Priority | Description |
|:---|:---|:---|
| **Cross-Channel Preview** | P0 | Visual verification of content across RCS, SMS, and WhatsApp. |
| **Template Management** | P1 | Central repository for approved marketing assets. |
| **AI Variation Lab** | P2 | Automatic generation of copy variations for A/B testing. |

## 6. Governance Agent (Safety Guard)
| Feature | Priority | Description |
|:---|:---|:---|
| **Frequency Control** | P0 | Global limits on per-user communication volume. |
| **Compliance Shield** | P1 | Monitoring of DND and regulatory exclusion lists. |
| **Bottleneck Analysis** | P2 | Identifying rules that are blocking the most potential volume. |
